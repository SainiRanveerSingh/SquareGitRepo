# Square Git Repositories – iOS App

# Git repo link
https://github.com/SainiRanveerSingh/SquareGitRepo.git

# Requirements
Display a scrollable list of repositories of “square” organisation in GitHub.
Each repository should present at least the name and description of the repo.
Use Swift and UIKit.

=================

This project demonstrates a clean and scalable iOS architecture using Swift and UIKit, following MVVM and SOLID principles. The app fetches and displays a list of repositories using native networking and structured UI components.

=================

This project demonstrates:

Clean MVVM architecture
Native networking with URLRequest
Reusable UI components (Loader, TableView, Cells)
JSON parsing using Codable
Unit testing with mocks
Dark mode support
Strong adherence to SOLID principles
